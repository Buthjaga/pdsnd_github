### Project Title
Bike Share Project

### Description
This project provides an easier interface for anyone to research Bike Sharing data
in the 3 Metro areas, Chicago, New York and Washington.

### Files used
The files used in this project are
* Bikeshare.py
* chicago.csv
* Washington.csv
* new_york_city.csv
